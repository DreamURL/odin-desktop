Privacy Policy

Last updated: March 8, 2026

Summary: Odin Desktop processes all data locally on your computer. We do not collect, store, or transmit any of your personal data, files, or search queries.

What Odin Does

Odin Desktop is a file search and document Q&A application that runs entirely on your local machine. It:

  • Indexes files on your local drives using NTFS MFT parsing
  • Runs a local AI model (LLM) on your CPU for document analysis
  • Stores its index database locally in your project directory

Data We Collect

None. Odin Desktop does not collect any data. Specifically:

  • No analytics or telemetry
  • No usage tracking
  • No error reporting to external servers
  • No account or registration required
  • No network requests to our servers (we don't have any)

Data That Stays on Your Machine

The following data is created and stored only on your local computer:

  • File index database — An SQLite database containing file names, paths, and metadata from your indexed drives
  • Search cache — A pickle cache file for faster search startup
  • AI model files — Downloaded LLM model weights stored in your app data directory
  • Application settings — Your preferences stored in localStorage within the Electron app

All of this data can be deleted at any time by uninstalling the application or removing the .odin_index directory.

Network Activity

Odin Desktop may make network requests only in these cases:

  • AI model download — When you choose to download an LLM model, it is fetched from Hugging Face or Ollama's model registry. This is a one-time download.
  • Ollama connection — If you configure Ollama as your LLM provider, Odin communicates with the Ollama server running on your local machine (localhost).

No file contents, search queries, or personal information are ever transmitted over the network.

Third-Party Services

Odin Desktop does not integrate with any third-party analytics, advertising, or tracking services.

Open Source

Odin Desktop is open source. You can inspect the complete source code on GitHub to verify these privacy claims.

Changes to This Policy

If this policy changes, the updated version will be posted on this page with a new "Last updated" date.

Contact
